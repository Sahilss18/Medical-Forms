# Medical Institution Recognition Portal - Frontend

## Overview

A modern, production-ready React application for the Medical Institution Recognition Portal. This system digitizes the Form 3F and licensing workflow for medical institutions applying for recognition to use essential narcotic drugs.

## Tech Stack

React 18 · TypeScript · Vite · React Router · TailwindCSS · Zustand · React Hook Form · Zod

## Getting Started

### Installation

```bash
# Install dependencies
npm install

# Set up environment
cp .env.example .env

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```
